## Getting Started

1. Clone the repository
```bash
git clone <your-repo-url>
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```

## Project Structure

- `src/` - Source code
  - `App.tsx` - Main application component
  - `features.md` - Feature comparison data
